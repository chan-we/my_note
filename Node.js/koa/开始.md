# 环境准备

node.js >= 7.6

npm >= 3.x

如果需要ts的话见[koa与ts](./koa与ts.md)

# 安装

```sh
npm install koa
```

## 脚手架

koa脚手架通过使用 koa 脚手架生成工具，可以快速创建一个基于 koa2 的应用的骨架

全局安装：

```
npm install koa-generator -g
```

创建项目：

```
koa2 koa_demo
```

# Hello World

```js
const Koa = require('koa')
const app = new Koa()

app.use( async ( ctx ) => {
  ctx.body = 'hello koa2'
})

app.listen(3000)
console.log('[demo] start-quick is starting at port 3000')
```

用`node`启动

```sh
node index.js
```
